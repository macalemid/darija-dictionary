Help build a dictionary!

Moroccan Arabic is one of the more difficult dialects of Arabic to learn. This is in part due to the fact that the dialect differs greatly from standard Arabic, but it is not helped by the fact that there are very few online resources to help learn the dialect. This dictionary aims to solve that problem.

You can search through all of the words that have already been added, and you can easily create and account to add and edit existing words.

At the moment the website only supports basic search functions and only has the Darija/MSA/English words together. However if there is a feature that you would like to see send us an email and we'll look into adding it! Hope you enjoy!
